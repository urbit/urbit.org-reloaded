These are a few of the top resources for getting started running your own instance of Urbit OS:
- [Get started locally](https://docs.urbit.org/get-on-urbit): Includes guidance for how to run with a free comet identity.
- [Groundseg](https://manual.groundseg.app/): Get started with Native Planet's urbit ship orchestration GUI.
- [Cloud hosting guide](https://docs.urbit.org/user-manual/running/cloud-hosting): Step-by-Step instructions to run in Digital Ocean, Hetzner, or Oracle clouds.
